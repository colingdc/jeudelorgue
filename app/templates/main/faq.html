FAQ

{% if current_user.is_manager() %}

  • Comment créer un tournoi ?

    Dans la barre de navigation, cliquez sur "Admin > Créer un tournoi". Renseignez un nom et une catégorie de tournoi, et une date de début pour le tournoi (date limite d'inscription).

    Cette date est uniquement indicative, pour le moment la fermeture des inscriptions se fait à la main par le manager du tournoi.

  • Comment créer le tableau du tournoi ?

    Sur la page du tournoi, cliquez sur "Créer le tableau".

    Remplissez un par un les joueurs du tableau en les sélectionnant dans la liste des joueurs disponibles. Vous pouvez spécifier un statut particulier (Q, WC, SE, LL) dans la 2ème colonne, et les numéros des têtes de série dans la 3ème.

    Si un joueur du tableau n'est pas disponible dans la liste, vous pouvez cliquer sur "Ajouter un joueur". La page permettant de créer un joueur s'ouvre dans un nouvel onglet. Créer alors tous les joueurs nécessaires.

    Une fois que c'est fait, revenez à l'onglet de création du tableau. Cliquez sur "Valider" en bas de la page pour sauvegarder l'état actuel du tableau. Vous êtes redirigé vers la page du tournoi. Cliquez alors sur "Modifier le tableau de départ", vous pourrez continuer à remplir le tableau, les joueurs que vous avez créés apparaitront dans la liste de joueurs disponibles.

  • Comment gérer les qualifiés ?

    Si certains joueurs du tableau de sont pas encore connus, laissez le champ "Joueur vide". Le nom sera automatiquement remplacé par "Qualifié N".

    Une fois le nom d'un qualifié connu, vous pourrez modifier le tableau de départ en conséquence. La modification sera prise en compte dans les tableaux des participants ayant déjà fait un pronostic.

  • Une fois le tournoi lancé, comment rentrer les résutats des matchs ?

    Sur la page du tournoi, cliquez sur "Mettre à jour le tableau".

    Cliquez sur le nom d'un joueur pour le faire avancer au tour suivant.

    Si vous avez cliqué par erreur sur un joueur, pas de panique, control-cliquez sur le nom du vainqueur pour effacer le contenu de la case.

  • Que faire à la fin du tournoi ?

    Lorsque le résultat du dernier match est validé, cliquez sur "Clôturer le tournoi".

    Les scores finaux seront calculés et les points ATP attribués aux participants.

{% endif %}

  • Comment participer à un tournoi ?

    Sur la page du tournoi, cliquez sur "M'inscrire".

  • Comment remplir mon tableau ?

    Une fois inscrit au tournoi, cliquez sur "Remplir mon tableau". Pour pronostiquer de chaque match, cliquez sur le nom du joueur qui va l'emporter selon vous. Une fois que vous avez pronostiqué tous les matchs, cliquez sur "Valider".
